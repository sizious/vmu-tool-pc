4. Tab overview

At the bottom of the main window, you should see 3 tabs with some informations. Please notify in all tabs you can use the context menu (popup menu) by clicking the right mouse button. The first tab's this one :

1) Details

It displays information on the selected file in the file view. You can modify fields (for a VMI only) and saving information by clicking the Save button.

2) Report

This tab show the full report (file information) on the selected file. You can save this report by selecting the Save report... item in the File menu.

3) Debug

All operations performed by VMU TOOL PC are shown here. You can clear it by selecting the Clear debug log... item in the View menu (or with right clicking on it).
