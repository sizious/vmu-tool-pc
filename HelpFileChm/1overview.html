1. Overview

Welcome to VMU TOOL PC ! We hope this application'll be pleasant to use and useful. This's a small Windows-Like explorer for Dreamcast save files. OK, it's very simple to use. Just double click on the VMU TOOL PC explorer. You should have this window :

Please notify a Windows XP theme is applied, so the appearance can be different if you are running an older Windows version. At the left, you can see a folder tree explorer, and at the top, a combo box. Here, you can select the folder you want (in the tree or in the combo box), where your saves are stored. If you know the full path to your saves, you can paste it in the combo box, and press enter. For example, my TEST folder contains my saves. I select it and I get this result :

As you see, files are automatically scanned and file count's updated. You can select the file you want to get information on it, at the bottom, Details tab sheet. In this screenshot, I selected 4 Wheel Thunder.

You can do some actions with the right click in this view. Click on an item with your mouse :

You can modify VMI, copy, move, delete and View in Hex editor the file, save VMI report (details) and refresh the view. Try it yourself ! For example, I select move to... :

Just pick your folder and click OK. Please notify this dialog's translated by Windows. You can right click too on the save icon, and everywhere. If you don't like the right click, you can find theses menus in the File menu.

As you see, using VMU TOOL PC is very easy !
