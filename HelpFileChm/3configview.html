3. Configuring the list view

You can easilly configure the file list. You can change place and hide/show columns, change icons size and change list view type disposition (details, small icons...). Everything of this can be done with the View menu.

1) Hiding / showing columns

Select this item (or hit Ctrl+L keys for the shortcut). You should have this dialog :

Just check the box with the column name you want to show. Please note that "Name" column can't be hidden. When you uncheck the box, the column'll be hidden after clicking the OK button. The old column position won't be saved, so when you want to show the column again, it will be added at the end of the list. You must move it with the next dialog, called "Moving columns".

2) Moving columns

After striking Ctrl+M keys or selecting the Move Columns... menu, this dialog's shown :

In this dialog, you can change position of the columns displayed. You can drag and drop items. When you have finished, just click the OK button and wait one second. You should see the columns moving in real time.

3) Changing icons size

Just select your prefered icon resolution in the Icon size menu, in the View menu.

4) Changing icons disposition

If you don't like the details view, you can choose another list view disposition type in the Style menu, located in the View menu.
