9. Console application

If you need it, you can launch the engine directly from the command line. Just add to your PATH environment variable the VMU TOOL PC path.

vmutool_pc {-dirs,-files,-infos,-checksum,-description,-copyright,-date,-resource,-name,-type,-copy,-size,-vmdesc,-dcdesc,-appname,-frames,-speed,-eyec_t,-crc,-datalen,-palette,-monoicon,-frame1,-frame2,-frame3,-eyec_pal,-eyec_bitmap,-bmpmonoicon,-bmpframe1,-bmpframe2,-bmpframe3,-bmpeyecatch,-w_check ,-w_desc ,-w_cpyrgt ,-w_date ,-w_rsrc ,-w_name ,-w_type ,-w_copy ,-w_sz ,-w_vmdesc ,-w_dcdesc ,-w_app ,-w_frms ,-w_spd ,-w_eyec_t ,-w_crc ,-w_datalen ,-w_pal ,-w_monoicon ,-w_frm1 ,-w_frm2 ,-w_frm3 ,-w_eyec_pal ,-w_eyec_bitmap ,-w_bmpmonoicon ,-w_bmpfrm1 ,-w_bmpfrm2 ,-w_bmpfrm3 ,-w_bmpeyec ,} [-o ]

: path of the directory/file.

Use one of the following arguments with :

*** Files and dirs scanning:
-dirs: display the list of directories found by scanning directory .
-files: display the list of files found by scanning directory .


*** Display VMI and VMS infos:
-infos: display the main infos from VMI file .


*** Display VMI infos:
-checksum: display the checksum from VMI file .
-description: display the description from VMI file .
-copyright: display the copyright from VMI file '.
-date: display the creation date from VMI file .
-resource: display the VMS resource name from VMI file .
-name: display the VM filename from VMI file .
-type: display the type from VMI file .
-copy: display the copy protection from VMI file .
-size: display the VMS file size in bytes from VMI file .


*** Display VMS infos:
-vmdesc: display the VM description from VMS file .
-dcdesc: display the DC description from VMS file .
-appname: display the application name from VMS file .
-frames: display the number of frames of color icon from VMS file .
-speed: display the animation speed of color icon from VMS file .
-eyec_t: display the type of eyecatch from VMS file .
-crc: display the CRC from VMS file .
-datalen: display the data length from VMS file .
-palette: display the color icon palette from VMS file .
-monoicon: display the monochrome icon from VMS file .
-frame1: display the 1st frame of color icon from VMS file .
-frame2: display the 2nd frame of color icon from VMS file .
-frame3: display the 3rd frame of color icon from VMS file .
-eyec_pal: display the eyecatch palette from VMS file .
-eyec_bitmap: display the eyecatch bitmap from VMS file .


*** Display VMS icons and eyecatch in BMP:
-bmpmonoicon: display the monochrome icon from VMS file in BMP format.
-bmpframe1: display the 1st frame of color icon from VMS file in BMP format.
-bmpframe2: display the 2nd frame of color icon from VMS file in BMP format.
-bmpframe3: display the 3rd frame of color icon from VMS file in BMP format.
-bmpeyecatch: display the eyecatch from VMS file in BMP format.


*** Write VMI infos:
-w_check : write the checksum in VMI file .
-w_desc : write the description in VMI file .
-w_cpyrgt : write the copyright in VMI file '.
-w_date : write the creation date in VMI file .
-w_rsrc : write the VMS resource name in VMI file .
-w_name : write the VM filename in VMI file .
-w_type : write the type of VMI file .
-w_copy : write the copy protection in VMI file .
-w_size : write the VMS file size in bytes in VMI file .


*** Write VMS infos:
-w_vmdesc : write the VM description in VMS file .
-w_dcdesc : write the DC description in VMS file .
-w_app : write the application name in VMS file .
-w_frms : write the number of color icon frames in VMS file .
-w_spd : write the animation speed of color icon in VMS file .
-w_eyec_t : write the eyecatch type in VMS file .
-w_crc : write the CRC in VMS file .
-w_datalen : write the data length in VMS file .
-w_pal : write the color icon palette in VMS file .
-w_monoicon : write the monochrome icon in VMS file .
-w_frm1 : write the 1st frame of color icon in VMS file .
-w_frm2 : write the 2nd frame of color icon in VMS file .
-w_frm3 : write the 3rd frame of color icon in VMS file .
-w_eyec_pal : write the eyecatch palette in VMS file .
-w_eyec_bitmap : write the eyecatch bitmap in VMS file .


*** Write VMS icons and eyecatch from BMP:
-w_bmpmonoicon : write the monochrome icon in VMS file from BMP data .
-w_bmpfrm1 : write the 1st frame of color icon in VMS file from BMP data .
-w_bmpfrm2 : write the 2nd frame of color icon in VMS file from BMP data .
-w_bmpfrm3 : write the 3rd frame of color icon in VMS file from BMP data .
-w_bmpeyec : write the eyecatch in VMS file from BMP data .


[-o ]: optionnal argument to save the output to file without CRLF conversion issues.
