6. Picture operations

You can perform nice picture operations. Here's listed how to do that and what's possible.

1) VMU icons

Right click on the VMU icon (on the Details tab). If your icon's animated, you should see this :

You can stop the animation with unchecking this box. If you select another file containing an animated icon, it'll be enabled again. After doing this, you can use this item to view a specific frame :

If your file have only one icon, it won't be animated, and the Display frames item'll be like this :
