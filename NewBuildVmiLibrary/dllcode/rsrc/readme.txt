I) Presentation : 
-----------------

BUILDVMI.DLL is a dynamic link library created in order to generate VMI files from VMS files.
VMS files are Dreamcast save files, and VMI a tiny header file, used in some proggies 
(DreamExplorer, VMU TOOL PC...).

II) Use : 
---------

This DLL is used in following proggies :
- VMU TOOL PC

Just replace the old library with this one.
